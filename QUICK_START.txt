==============================================

1. Open terminal in backend folder:
   cd backend

2. Install dependencies (if not already done):
   npm install

3. Create .env file:
   Copy .env.example to .env
   
   Windows:
   copy .env.example .env
   
   Linux/Mac:
   cp .env.example .env

4. Edit .env file:
   - Change JWT_SECRET to something random
   - Update MONGO_URI if needed

5. Start MongoDB (if not running):
   - Ensure MongoDB is running on localhost:27017
   - Or update MONGO_URI in .env

6. Start backend:
   npm start
   
   Expected output:
   "Server running on port 5000"
   "MongoDB connected: 127.0.0.1"

ğŸ› TROUBLESHOOTING
==============================================

PROBLEM: Backend won't start
SOLUTION: Check MongoDB is running

PROBLEM: Teacher dropdown is empty
SOLUTION: Create at least one teacher first

PROBLEM: Can't login
SOLUTION: 
- Check backend is running
- Check console for errors
- Clear browser localStorage

PROBLEM: Profile dropdown doesn't work
SOLUTION:
- Hard refresh browser (Ctrl+F5)
- Check console for JS errors

PROBLEM: Socket.io errors in console
SOLUTION:
- These are normal warnings
- Backend doesn't have socket.io server
- App works fine without it
